argmax-c             -     Returns the indices of the maximum values along an axis.   
nanargmax-c          -     Return the indices of the maximum values in the specified axis ignoring NaNs.
argmin-c             -     Returns the indices of the minimum values along an axis.
nanargmin-c          -     Return the indices of the minimum values in the specified axis ignoring NaNs.
argwhere-c           -     Find the indices of array elements that are non-zero, grouped by element.
nonzero-c            -     Return the indices of the elements that are non-zero.
flatnonzero        -     Return indices that are non-zero in the flattened version of a.
where-c              -     Return elements chosen from x or y depending on condition.
searchsorted       -     Find indices where elements should be inserted to maintain order.
extract-c            -     Return the elements of an array that satisfy some condition.
